# Car Buying Prediction

This project predicts whether a person will buy a car using a machine learning classification model.

## Dataset Features
- Salary
- Age
- Married
- Car
- Loan

Target variable:
- Buy (0 = No, 1 = Yes)

## Algorithm Used
Gaussian Naive Bayes

## Steps
1. Load dataset
2. Select features and target
3. Train the model
4. Make predictions

## Technologies Used
- Python
- Pandas
- Scikit-learn
